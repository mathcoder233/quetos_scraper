# Quotes Scraper

A Python scraper that extracts quotes, authors, and tags from [quotes.toscrape.com](https://quotes.toscrape.com), handles login and CSRF tokens, supports pagination, and saves the collected data to an Excel file.

## Features

- Handles login and CSRF tokens  
- Supports pagination (scrape multiple pages)  
- Saves data to Excel (.xlsx)  
- Modular structure: session.py, parser.py, excel_saver.py, main.py  
- Fast and easy to extend for other websites
